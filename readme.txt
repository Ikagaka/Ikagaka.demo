Ikagaka.js demo を任意のサイトに置けるようにした版
===============================================

あなたのサイトでゴーストが動きます。

イカガカ demo
http://ikagaka.github.io/Ikagaka.demo/

## ダウンロード

ページ右側の緑のボタン「Clone or download」をクリックし、さらに下に出てきた「Download ZIP」をクリックしてダウンロードして下さい。

## 使用方法

1. js/ikagakaConfig.jsを説明に合わせて設定します。
2. vendor/narに起動したいnarを置きます。（元々あったnarは削除してかまいません。）
3. index.html, js, css, vendorをこのままの配置でサイトの任意の場所に置きます。（index.htmlは名前変更可能です。）

## 注意

- http://ikagaka.github.io/Ikagaka.demo/ の「対応状況」に記述してあるとおりの仕様になっています。
- 次期バージョンのイカガカは別の仕組みになるので、このライブラリ自体が更新される予定はありません。
